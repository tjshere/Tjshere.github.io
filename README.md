# Torren Moore Jr. - Professional Portfolio

## Project Description
Welcome to my professional portfolio website! This site showcases my skills, experience, and projects as an IT professional and Navy Veteran. It serves as a digital resume and a platform to connect with potential employers and collaborators.

## Features
- **Responsive Design**: The website is designed to be fully responsive, ensuring a seamless experience on various devices, including desktops, tablets, and smartphones.
- **Contact Form**: A user-friendly contact form allows visitors to reach out to me directly for inquiries or collaboration opportunities.
- **Project Showcase**: Highlights of my key projects and accomplishments, demonstrating my technical skills and professional journey.
- **About Me Section**: A brief introduction that outlines my background, skills, and values as a professional.

## Getting Started

### Prerequisites
- [XAMPP](https://www.apachefriends.org/index.html) installed.

### Hosting Locally with XAMPP
1. **Download and Install XAMPP**:
   - Visit the XAMPP website and download the version compatible with machine
   - Follow the installation instructions to set up XAMPP on your machine.

2. **Start XAMPP**:
   - Open the XAMPP application.
   - Start the Apache server by clicking the "Start" button next to "Apache".

3. **Prepare Your Project**:
   - Navigate to the XAMPP `htdocs` directory. This is usually located at `/Applications/XAMPP/htdocs`.
   - Create a new folder for your project, e.g., `torren-portfolio`.

4. **Add Your Files**:
   - Copy your `index.html` and `styles.css` files into the new project folder you just created.

5. **Access Your Website**:
   - Open a web browser and type `http://localhost/torren-portfolio` in the address bar.
   - You should see your portfolio website live!
